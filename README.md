# SLAM-Robot
A robot that uses cameras to map out a given area then is able to go to anywhere directed on the created map.

I decided to make this project as being able to map out a room seemed really cool. I also saw a video on how newer cars could implement lidar technology for auto driving, and I found that very cool and wanted to try it out.
